## Proposed Test Cases

The following test cases are planned for system testing after implementation:

| Test Case ID | Description | Expected Result |
|---------------|-------------|-----------------|
| TC1 | User Registration | New user account should be created successfully |
| TC2 | User Login | User should log in with valid credentials |
| TC3 | Browse Menu | Menu items should display correctly |
| TC4 | Place Order | Order should be saved successfully |
| TC5 | Track Order | Correct order status should be displayed |
| TC6 | Rate Order | Rating should be stored successfully |
| TC7 | Staff View Orders | Staff should view all incoming orders |
| TC8 | Update Order Status | Status should update correctly |
| TC9 | Admin Manage Menu | Menu updates should save successfully |
| TC10 | View Reports | Reports should display correctly |

These test cases will be used during the implementation and testing phase of the Zahib system.

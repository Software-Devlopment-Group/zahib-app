# Zahib App

Smart Food Ordering and Queue Management System

## Project Description
Zahib App is a smart food ordering system designed to improve customer experience and reduce waiting time through digital ordering, queue management, and real-time order tracking.

## Features
- User Registration & Login
- Food Ordering System
- Queue Management
- Order Tracking
- Rating System
- Staff Panel
- Data Analytics

## Technologies Used
- Python
- Tkinter
- SQLite
- Matplotlib
